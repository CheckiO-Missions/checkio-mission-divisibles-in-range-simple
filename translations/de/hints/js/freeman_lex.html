Math.floor(n / k)
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur das Ergebnis des Ausdrucks zurückgeben und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

function countDivisible(n: number, k: number): number {

    return Math.floor(n / k)
}

Klar, schnappen Sie es sich!

Iterieren Sie über den Bereich von k bis n mit einem Schritt von k und erhöhen Sie einen vordefinierten Zähler. 

function countDivisible(n: number, k: number): number {

    let count = 0;
    for (let i=k; i < n + 1; i+=k) count += 1;
    
    return count;
}
