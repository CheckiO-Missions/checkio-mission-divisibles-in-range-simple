Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Rozwiązanie jest bardzo proste: wystarczy znaleźć wynik dzielenia podłogowego dywidendy (n) i dzielnika (k).

Ok, od czego powinienem zacząć?

Proszę poczytać o metodzie floor division i z niej skorzystać! 

Math.floor(n / k)
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zwrócić wynik wyrażenia i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

function countDivisible(n: number, k: number): number {

    return Math.floor(n / k)
}

Jasne, proszę pobrać!

Proszę wykonać iterację po zakresie od k do n z krokiem k i zwiększyć predefiniowany licznik. 

function countDivisible(n: number, k: number): number {

    let count = 0;
    for (let i=k; i < n + 1; i+=k) count += 1;
    
    return count;
}
