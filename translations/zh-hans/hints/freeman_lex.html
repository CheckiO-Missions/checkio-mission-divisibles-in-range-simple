阅读floor division 运算符并使用它！ 

n // k
那么，我的最终解决方案应该是怎样的呢？

你只需返回表达式的结果，并将代码封装成函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def count_divisible(n: int, k: int) -> int:

    return n // k

当然，抓紧时间！

建立一个从k 到n 的整数范围，包括超过k 的步长，并找出这个范围的长度（其中的元素个数）。 

def count_divisible(n: int, k: int) -> int:

    return len(range(k, n+1, k))

从标准模块导入特定函数。 

from operator import floordiv as count_divisible
